nvm控制node版本号

Created At : 2020-06-24 16:18
Views 👀 :
  1. 安装
  2. 切换node版本

安装

  • 访问 https://github.com/coreybutler/nvm-windows/releases, 下载最新版本 nvm-setup.zip ,解压安装
  • 安装过程中，选择 node 不同版本的安装目录（尽量避开 C: 盘，以免时间过长后，C: 盘空间不够用），举例：nvm 安装目录为 E:\software\nvm
    • 安装过程中会让选择 nodejs 目录，经验证，此时选择的 nodejs 目录其实是，npm 运行目录，举例选择 E:\software\Nodejs
    • 经安装不同 nodejs 版本且切换不同版本后，发现 E:\software\Nodejs 其实仅仅是当前切换后 nodejs npm 安装目录的快捷方式。 故安装时特意选择 nodejs 目录并未达到所有 nodejs 版本共用同一个 npm 环境的目的；鉴于不同的 nodejs 支持的 npm 里面库的版本很有可能有差异，故我们考虑做公共的 npm 运行目录，按需引入即可；
  • cmd 里面 nvm list available，查看可选择 nodejs 版本；执行 nvm install 版本号，安装对应版本的 nodejs

切换node版本

  • 上一步仅仅是安装了不同版本的 nodejs ,真正切换使用需要执行 nvm use 版本号，来到达切换环境的目的
