通过github搭建自己的博客

  • 工具：nvm、node、npm、yarn、vscdode、git
  • 系统：win10

安装nvm

  • https://github.com/coreybutler/nvm-windows/releases, 下载最新版本 nvm-setup.zip ,解压安装
  • 安装过程中，选择 node 不同版本的安装目录（尽量避开 C: 盘，以免时间过长后，C: 盘空间不够用），举例：nvm 安装目录为 E:\software\nvm， 安装过程中会让选择 nodejs 目录，经验证，此时选择的 nodejs 目录其实是，npm 运行目录，举例选择 E:\software\Nodejs，经安装不同 nodejs 版本且切换不同版本后，发现 E:\software\Nodejs 其实仅仅是当前切换后 nodejs npm 安装目录的快捷方式。故安装时特意选择 nodejs 目录并未打到所有 nodejs 版本共用同一个 npm 环境的目的；鉴于不同的 nodejs 支持的 npm 里面库的版本很有可能有差异，故我们考虑做公共的 npm 运行目录，按需引入即可；
  • cmd 里面 nvm list available，查看可选择 nodejs 版本；执行 nvm install 版本号，安装对应版本的 nodejs
  • 上一步仅仅是安装了不同版本的 nodejs ,真正切换使用需要执行 nvm use 版本号，来到达切换环境的目的
  • github 上新建 repository （ 以 github用户名.github.io 为仓库名称 ）, nvm use 版本号，切换 nodejs 到最新版本；
  • 利用 hexo-cli 快速生成模板项目： npm i -g hexo-cli ; hexo init 项目名；cd 项目目录
  • hexo new “hello world” ； hexo generate ；本地启动：hexo server ；
  • 项目下 _config.yml 添加：
  • push到仓库并在github上发布更新：hexo clean ； hexo deploy ；
  • 访问 https://github用户名.github.io 即为自己的博客首页地址
